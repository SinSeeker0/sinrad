BUNDLED MUSIC
=============
Drop the tracks you want SHIPPED with the app (included in the GitHub release) into THIS folder.
On first run they are copied into the player folder so they show up in the playlist.

- Supported: mp3, ogg, wav, m4a, flac, webm, opus
- Tracks you add later via the in-app folder button go into the PLAYER folder (shown in the
  music panel when empty), NOT here, and they are never overwritten.
- If you delete a bundled track from the player folder, it is restored from here on next launch.
